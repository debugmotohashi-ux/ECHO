ECHO v2 — 面談・教育 共有ハブ(受け皿構成)
==============================================

■ 構成の変更
録音と文字起こしは、OS純正アプリ(Pixel: レコーダー / iPhone: ボイスメモ)に
全面委任しました。ECHOは受け取ったテキストの
「フォーマット整形(本橋式)→ChatGPT連携→保存→デバイス間同期」に専念します。

■ アップロード手順
以下の3ファイルをGitHubのリポジトリに上書きアップロードしてください:
  1. index.html          … アプリ本体(v2に全面刷新)
  2. manifest.webmanifest … 共有ターゲット(共有先にECHOを表示)を追加
  3. sw.js               … キャッシュ更新(v2)
アイコン3種は変更なし(そのままでOK)。

■ Pixelでの使い方
0. 【重要・初回のみ】共有ターゲットを有効にするため、既存のECHOアプリを
   一度アンインストール→ChromeでURLを開き直し→「アプリをインストール」で再インストール
   (マニフェストの変更はアプリの再インストールで確実に反映されます)
1. レコーダーアプリで録音(設定→文字起こし言語: 日本語)
2. 録音を開く→共有→「文字起こし」→共有先で「ECHO」を選択
3. ECHOが開き、テキストが入った状態に→面談/教育を選択→「ChatGPTへ」

■ iPhoneでの使い方
1. ボイスメモで録音(文字起こしはリアルタイム表示可)
2. 文字起こしを全選択してコピー
3. ECHOを開く→「貼り付け」ボタン→面談/教育を選択→「ChatGPTへ」

■ ボタンの動き
- 保存(履歴・同期): テキストを履歴とSupabaseに保存
- ChatGPTへ: 保存+本橋式プロンプト付きコピー+ChatGPT起動を1タップで実行

■ Supabase同期
設定方法は従来と同じ(歯車→URL/Anon Key→ログイン)。
既存の設定・ログイン状態・過去の記録はそのまま引き継がれます。
