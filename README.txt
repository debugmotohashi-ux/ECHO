ECHO - 録音手帳
================

■ 使い方
1. ECHO.html を iPhone の Safari で開いてください(プレビュー画面ではマイクが使えません)
2. 録音ボタンを押すとマイク許可を求められるので「許可」をタップ
3. 動作確認できたら、Safariの共有ボタン→「ホーム画面に追加」でアイコン付きアプリとして使えます

■ 同梱ファイル
- ECHO.html      … アプリ本体(アイコン埋め込み済み・単体で動作)
- icon-1024.png  … アイコン原寸(予備)
- icon-192.png   … アイコン(予備)
- icon-180.png   … アイコン(予備)

■ v2で追加された機能
1. 画面スリープ防止(Wake Lock)
   録音中はiPhoneの画面が自動ロックされなくなります。長時間録音でも録音が途切れません。
   ※録音中はアプリを前面に表示したままにしてください。

2. クラッシュ復元
   録音中の音声は数秒ごとに端末内(IndexedDB)へ自動退避されます。
   万一ブラウザが落ちても、次に開いたときに「未完了の録音があります」と表示され、
   未処理の音声を高精度モデルで再処理して復元できます。
   確定済みのチャンクの音声データは自動削除されるため、容量は増え続けません。

3. Nexus自動連携(Capture層)
   確定した文字起こしは、履歴とは別に「nexus_capture」という共有データベース(IndexedDB)へ
   自動保存されます。保存形式:
   { id, ts, source:"ECHO", layer:"capture", type:"voice_transcript",
     scope, format, durationSec, text }

   【重要】Nexusアプリがこのデータを直接読むには、ECHOとNexusを
   「同じドメイン」で配信する必要があります(例: 両方とも同じGitHub Pages上に置く)。
   ローカルファイル(file://)で開いた場合や別ドメインの場合は共有されないため、
   その間は「JSON書き出し」ボタンでエクスポートしたファイルをNexusに取り込んでください。

4. 幻覚フィルタ・無音スキップ
   Whisperが無音時に出しがちなジャンクテキスト(「ご視聴ありがとうございました」等)を
   自動除去し、無音セグメントは速報処理をスキップして電池消費を抑えます。

■ マルチデバイス同期(Supabase)
右上の歯車アイコンから設定すると、iPhone・iPad・Androidの間で記録が同期されます。
セットアップ手順は同梱の SUPABASE_SETUP.txt を参照してください。
未設定でも端末内保存のみで全機能が使えます(オフライン時は「未同期」として蓄積され、オンライン復帰時に自動同期)。

■ データの保存場所
確定した文字起こしはすべて端末内に保存されます(外部送信なし)。
- 履歴(直近30件): アプリ内の「これまでの記録」
- Nexus用アーカイブ(全件): 端末内の共有データベース「nexus_capture」
将来NexusをECHOと同じドメインに置いた時点で、Nexusはこのアーカイブを直接読み出せます。
データを長期で確実に残すため、「ホーム画面に追加」した状態での利用を推奨します。
定期的な「JSON書き出し」がバックアップにもなります。

■ 注意点
- 初回起動時にWhisperモデル(tiny + large-v3-turbo、合計1.5GB前後)をダウンロードします。
  Wi-Fi環境での初回起動を推奨します
- 録音・文字起こしはすべて端末内で完結し、外部サーバーへの送信は一切ありません
